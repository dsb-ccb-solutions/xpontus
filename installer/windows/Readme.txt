[ XPontus 1.0.0RC1 - Readme ]
> 1 - A quick introduction <
XPontus is a free XML Editor for text editing. 
XPontus is totally independant from the Operating System which runs it. 
The XPontus homepage is http://xpontus.sourceforge.net/ . 
You can contact the author : yveszoundi@users.sf.net .   

> 2 - The documentation <
A documentation is available in the doc folder. 
It was generated with Docbook XML under GNU-Linux.   

> 3 - Licencing issues <
The licences used by XPontus is GPL. 
You can find a copy under the installation folder.
