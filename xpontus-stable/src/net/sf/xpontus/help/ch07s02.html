Using Batch Validation

You can validate a batch of XML files(and or directories) using XML Batch Validation.

The menu is Tools->XML Batch Validation.

By default, the Batch Validation will be performed on file which filenames end by ".xml". You can choose to add or remove file patterns(*.svg,*.xsl,etc.)

If the option Look into sub-directories is selected, all the subdirectories will be scanned according to the XML patterns(file extensions) you specified.
