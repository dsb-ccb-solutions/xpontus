Introduction

We can define XSLT scenarios as XSLT transformations

The user defines the following data :

  • an XML input(a XML file)
  • an XSLT stylesheet which will be used to transform the XML file
  • the type of result expected (svg, html, pdf, text, xml)
  • the output file to store the result
